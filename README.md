# Tic Tac Toe

## 게임 설명
* 3X3 크기의 판에서 상대보다 먼저 가로, 세로 대각선 중 하나 이상의 연속된 3개문양을 만들면 승리한다.

## 사용 방법
* 메뉴에서 1인, 2인 모드 선택 가능.
* 1P가 돌을 선택(O 또는 X)하고 2P는 자동으로 남는 돌 선택
* 화살표와 스페이스바로 원하는 위치에 돌을 놓을 수 있다.
